Внешние устройства

Внешние (периферийные) устройства предназначены для выполнения вспомогательных операций. Благодаря им компьютерная система приобретает гибкость и универсальность.

Внешние устройства разделяют на группы:

  1. Ввод данных:
    • Клавиатура - устройство ввода знаковых данных
    • Мышь - перемещение курсора путем передвигания мыши по поверхности
    • Тачпад - сенсорная пластина, реагирующая на движение пальца по поверхности
    • Геймпад - Представляет собой пульт, который удерживается двумя руками, для контроля его элементами управления используются большие пальцы рук. Используются для игр
    • Сканер - устройство, которое, анализируя какой-либо объект (обычно изображение, текст), создаёт его цифровое изображение
    • Графический планшет - устройство для ввода информации созданной от руки непосредственно в компьютер. Состоит из пера и плоского планшета, чувствительного к нажатию или близости пера
    • Веб-камера - цифровая теле- или фотокамера, способная в реальном времени фиксировать изображения, предназначенные для дальнейшей передачи по сети Интернет
  2. Вывод данных:
    • Монитор - оборудование для визуального отображения текстовой и графической информации, преобразует цифровую и (или) аналоговую информацию в видеоизображения
    • Принтер - оборудование для печати документов на бумаге или прозрачном носителе
    • Колонки / наушники - оборудование для воспроизведения звука
  3. Хранение данных:
    • Flash-накопители / внешние диски - запоминающие устройства, использующие в качестве носителя или флэш-память, или внешний жесткий диск, подключаемые к компьютеру или иному считывающему устройству по интерфейсу USB
